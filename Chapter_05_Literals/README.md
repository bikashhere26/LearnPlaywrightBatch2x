# Chapter 05: Literals

This folder contains JavaScript examples for literal values and string formatting.

Files:
- `22_Literal.js` - basic literal values
- `23_null_undefined.js` - `null` vs `undefined`
- `24_null.js` - examples using `null`
- `25_Literal_All.js` - overview of literal types
- `26_literal_number_All.js` - JavaScript number literal support, including `Number` and `BigInt`
- `27_string.js` - string literals
- `28_Template_Literal.js` - template literals with interpolation
- `29_Backtick_single_double.js` - comparison of backticks, single quotes, and double quotes

## Run examples

Use Node.js to execute any file:

```bash
node 29_Backtick_single_double.js
```
