# Chapter 04: JavaScript Concepts

This folder contains JavaScript examples covering variable declarations, scope, and hoisting.

Files:
- `09_var_let_const.js` - `var`, `let`, and `const` overview
- `10_function.js` - function basics
- `11_var_explained.js` - deep dive into `var`
- `12_let_explained.js` - deep dive into `let`
- `13_const_explained.js` - deep dive into `const`
- `14_var_functionscope.js` - function scope with `var`
- `15_let_scope.js` - block scope with `let`
- `16_Hoisting.js` - hoisting concept overview
- `17_hoisting_fn.js` - function hoisting
- `18_let_hoisting.js` - `let` and hoisting
- `19_let_hoisting_block.js` - block hoisting behavior
- `20_let_const.js` - `let` vs `const` examples
- `21_Jr_Qa.js` - junior QA practice questions

## Run examples

Use Node.js to execute any file:

```bash
node 09_var_let_const.js
```
