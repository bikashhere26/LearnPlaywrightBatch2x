# Chapter 1: Basics & Setup Verification

Welcome to **Chapter 1**! This chapter is focused on getting your environment set up and verifying that JavaScript (Node.js) runs correctly on your machine.

## Files Included

- `01_Basics.js`: A simple introductory script to test basic output.
- `02_JS.js`: A quick look at running JavaScript from the terminal.
- `03_JS_Verify_Setup.js`: A script designed to verify that your Node.js setup is functioning as expected for upcoming Playwright exercises.

## How to Run

Navigate to the root directory and execute the files using Node.js:

```bash
node Chapter_01_Basics/01_Basics.js
node Chapter_01_Basics/03_JS_Verify_Setup.js
```

Ensure that you see the expected console output without any errors. This confirms your environment is ready for the next chapters.
