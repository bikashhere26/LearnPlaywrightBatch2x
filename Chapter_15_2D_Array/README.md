# Chapter 15: 2D Arrays

This chapter introduces two-dimensional arrays in JavaScript, including how to create arrays of arrays, traverse them with nested loops, and use them for matrix-style problems and pattern printing.

## Files

- `138_2D_Array.js`
- `139_2d.js`
- `140_REAL.js`
- `141_2d_Array_Fn.js`
- `142_IQ_Right_Pattern_Py.js`
- `Cheatsheet.md`

## Topics

- 2D array creation and indexing
- Nested loops with rows and columns
- Iterating over arrays of arrays
- Building and printing star patterns
- Function-based 2D array handling
- Quick reference notes in the cheatsheet

## Running examples

Use Node.js from the repository root. For example:

```bash
node Chapter_15_2D_Array/138_2D_Array.js
```
