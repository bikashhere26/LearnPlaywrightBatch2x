# Chapter 13: Strings

This chapter covers JavaScript string fundamentals, including string properties, search methods, substring extraction, transformation, and conversion.

## Files

- `118_Strings.js`
- `119_String_Properties.js`
- `120_Search_Check_Str.js`
- `121_Substring.js`
- `122_Transform_Str.js`
- `123_String_Conversion.js`

## Topics

- String literals and primitive behavior
- `String` properties and length
- Searching inside strings with `indexOf`, `includes`, and related methods
- Extracting substrings with `substring`, `slice`, and `substr`
- Transforming strings with `toUpperCase()`, `toLowerCase()`, `trim()`, and replacement techniques
- Converting values to strings and mapping string values in JavaScript

## Running examples

Use Node.js to run each file from the repository root. For example:

```bash
node Chapter_13_Strings/118_Strings.js
```
