# Chapter 22 — Encapsulation

This chapter demonstrates the encapsulation concept in JavaScript using examples.

Files included:

- `179_Ecap.js` — basic encapsulation example
- `180_REAL_EXAMPLE.js` — real-world encapsulation use-case
- `181_Ecap_Car.js` — encapsulation with a Car example
- `182_ECap_Bank.js` — bank account encapsulation example

How to use

1. Open the example file you want to run (for example, `182_ECap_Bank.js`).
2. Run it with Node.js:

```
node Chapter_22_Encapsulation/182_ECap_Bank.js
```

Notes

- These examples are standalone JavaScript files intended for learning and experimentation.
