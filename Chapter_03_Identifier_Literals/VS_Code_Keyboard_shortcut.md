# Visual Studio Code Keyboard Shortcuts (Windows)

Here is a comprehensive list of the most useful default keyboard shortcuts for Visual Studio Code on Windows, organized by category.

### General
| Action | Windows Shortcut |
| :--- | :--- |
| Show Command Palette | `Ctrl + Shift + P` or `F1` |
| Quick Open, Go to File | `Ctrl + P` |
| New window/instance | `Ctrl + Shift + N` |
| Close window/instance | `Ctrl + Shift + W` |
| User Settings | `Ctrl + ,` |
| Keyboard Shortcuts | `Ctrl + K` `Ctrl + S` |
