# Chapter 12: Functions

This chapter covers JavaScript functions, including declaration, parameters, return values, and common function patterns.

## Files Included

- `96_Functions.js` - Basic function declaration and return values
- `97_Type1_Fn_Basic_Functions.js` - Simple function examples and direct usage
- `98_Type2_Fn_With_Param_No_Return.js` - Function with parameters and no explicit return
- `99_Type3_Fn_without_Param_Return_Type.js` - Function without parameters but with return value
- `100_Type4_Fn_With_Param_With_Return.js` - Function with parameters and a return value
- `101_Template_literal.js` - Using template literals inside functions
- `102_Fn_Expression.js` - Function expressions and assignment to variables
- `103_Arrow_Fn.js` - Arrow function syntax and concise returns
- `104_Arrow_Fn_REAL.js` - Real arrow function use cases
- `105_IIFE.js` - Immediately invoked function expressions
- `106_Default_Param_Fn.js` - Default parameter values
- `107_IQ.js` - Practice and interview examples
- `108_Rest_Param_Fn.js` - Rest parameter usage in functions
- `109_IQ.js` - Practice and interview examples
- `110_Spead_IQ.js` - Spread operator examples with functions
- `111_Scope._Fn.js` - Function scope and variable visibility
- `112_IQ.js` - Practice and interview examples
- `113_Closure.js` - Closure examples and behavior
- `114_Closure.js` - Closure practice scenarios
- `115_API_REAL_Closure.js` - Real closure example with API-style usage
- `116_Higher_Order_Fn.js` - Higher-order function examples
- `117_Pure_Fn.js` - Pure function examples

## Topics Covered

- Defining and calling functions
- Parameters and arguments
- Return values and output
- Function expressions and arrow functions
- Immediately invoked functions (IIFE)
- Default parameters and rest/spread usage
- Scope, closures, higher-order functions, and pure functions
- Function behavior and examples
