# Chapter 23: Inheritance

This chapter covers JavaScript inheritance concepts using class-based syntax and prototype behavior.
It shows how a child class can reuse parent class code and how overridden methods work.

## Topics covered

- Single inheritance
- Multilevel inheritance
- Hierarchical inheritance
- `super` keyword usage
- Prototype-based inheritance behavior
