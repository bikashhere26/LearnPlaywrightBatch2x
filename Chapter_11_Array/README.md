# Chapter 11: Arrays

This chapter covers JavaScript arrays, including creation, access, modification, iteration, searching, and transformation.

## Files Included

- `83_Array.js` - Basic array creation and usage
- `84_Arrays.js` - Array examples and operations
- `85_Access_Array.js` - Accessing array elements and indices
- `86_Arrays_Adding_Remove.js` - Adding and removing items from arrays
- `87_Adding_Remove2.js` - More add/remove methods and examples
- `88_REAL_Example.js` - Real-world array example with practical usage
- `89_Searching.js` - Searching arrays and finding elements
- `90_Iterate.js` - Iterating arrays using loops and iteration methods
- `91_Transform_Array.js` - Transforming arrays with map, filter, and other techniques

## Topics Covered

- Array creation and indexing
- Accessing and updating array elements
- Adding and removing items from arrays
- Iterating arrays with loops
- Searching and filtering arrays
- Transforming arrays with higher-order methods

## Run examples

Use Node.js to execute any file:

```bash
node 83_Array.js
```
