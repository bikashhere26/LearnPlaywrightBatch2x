# Chapter 09: User Input

This chapter covers different methods of accepting user input in JavaScript applications.

## Files

- **68_User_Input.js** - Basic user input methods and concepts
- **69_Node_readline.js** - Using Node.js readline module for synchronous user input
- **70_prompt_sync.js** - Using prompt-sync library for user input handling

## Topics Covered

- User input methods in Node.js
- readline module basics
- prompt-sync library
- Synchronous vs asynchronous input
- Handling user responses
- Input validation and processing
- Real-world input handling scenarios

## Run examples

Use Node.js to execute any file:

```bash
node 68_User_Input.js
```

## Prerequisites

For prompt-sync examples, install the required package:

```bash
npm install prompt-sync
```
